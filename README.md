# DoIt - Task Management Web App

DoIt is a modern and responsive task management web application that helps users organize, track, and manage their daily tasks efficiently.

The application provides a clean dashboard where users can create tasks, monitor progress, manage categories, and customize settings.

---

## Features

- User Authentication (Login / Signup)
- Task Creation and Management
- Task Status Tracking
- Task Categories (Work, Personal, Study, Fitness)
- Daily Progress Tracking
- Dashboard with Statistics
- Light Mode / Dark Mode
- Profile Settings
- Security Settings
- Data Export
- Delete Account / Data Management
- Responsive UI Design
- Multiple Views (List, Kanban, Calendar)

---

- Total Tasks Counter
